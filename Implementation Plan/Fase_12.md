## FASE 12 — MANTENIMIENTO MENSUAL

### Tareas semanales (cada semana)

Publicar un artículo nuevo en el blog:
1. Crear el archivo .md en src/content/blog/ con el frontmatter correcto
2. Redactar el contenido con mínimo 1.000-1.500 palabras siguiendo la estructura 
de H2 definida
3. Incluir mínimo 3 enlaces internos a otras páginas del sitio
4. git add src/content/blog/nombre-articulo.md
5. git commit -m "blog: añadir artículo [título del artículo]"
6. git push origin main
7. GitHub Actions hace el build y despliegue automáticamente en 2-3 minutos

Actualizar Google Business Profile:
- Crear una publicación nueva con: novedad de la convocatoria, consejo de 
estudio, enlace a un artículo del blog o vídeo de YouTube reciente

Responder reseñas de Google:
- Responder a cualquier nueva reseña que haya llegado durante la semana. Todas 
las reseñas deben tener respuesta.

### Tareas mensuales (cada mes)

Google Search Console — revisar estas secciones:
- Rendimiento: qué consultas generan más clics e impresiones. Identificar 
keywords en posición 5-15 que con más contenido podrían subir al top 3
- Cobertura: verificar que no hay páginas con errores de indexación. Si hay páginas 
con estado "Excluidas", investigar por qué
- Core Web Vitals: verificar que no hay regresión en los indicadores de rendimiento
- Datos estructurados: verificar que el schema markup sigue sin errores

Google Analytics 4 — revisar:
- Usuarios únicos del mes: ¿crecen mes a mes?
- Canal de adquisición: ¿crece el tráfico orgánico (búsqueda)?
- Página más visitada: ¿qué contenido atrae más?
- Conversiones: ¿cuántos formularios se han enviado?

Actualizar artículos si hay cambios de convocatoria:
- Cuando salga un nuevo BOE con datos actualizados (número de plazas, fecha de 
examen), actualizar inmediatamente los artículos y páginas que mencionen esos 
datos
- Especialmente: la página de SS si el examen del 28 junio de 2026 tiene algún 
cambio

### Tareas trimestrales

oposiciones AGE", "preparar oposiciones administrativo estado online", 
"oposiciones seguridad social preparadora", "preparar AGE y seguridad social a la 
vez"
- Anotar en qué posición aparece mcoposiciones.com para cada una
- Si alguna keyword está en posición 5-15, reforzar esa página con más contenido y 
más enlaces internos

enlace correcto a la web
